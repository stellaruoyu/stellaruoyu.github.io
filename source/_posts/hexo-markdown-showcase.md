# Hexo Markdown 语法展示

欢迎来到这篇展示文章！在这里，我们将展示Hexo支持的各种Markdown语法特性。

## 1. 基本文本格式

### 1.1 强调语法

- **粗体文本**
- *斜体文本*
- ***粗斜体文本***
- ~~删除线文本~~

### 1.2 引用

> 这是一个引用示例
>> 这是嵌套引用

### 3.1 行内代码

```javascript

}

console.log(greeting('Hexo'));
```

| 功能 | 语法 | 示例 |
